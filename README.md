# ArcShelf

The missing bookmark experience for Arc Browser.
A beautifully designed bookmark manager with a native sidebar, lightning-fast search, smooth animations, and keyboard-first workflows.

---

## ✨ Features

- 📚 Beautiful native sidebar
- 🔍 Instant fuzzy search
- 🗂 Unlimited nested folders
- ✨ Drag & Drop organization
- ⌨️ Keyboard-first experience
- 🌙 Light, Dark & System themes
- 📦 Import & Export bookmarks
- 🎨 Colored folders
- 🖼 Website favicons
- ⚡ Built for performance

# 🚀 Installation

## Install from Release

1. Download the latest release.
2. Extract the ZIP.
3. Open:

```
chrome://extensions
```

4. Enable **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted ArcShelf folder.
7. Press **⌘ + Shift + B** to open ArcShelf.

---

## Build from Source

```bash
git clone https://github.com/Sakthi51/ArcShelf.git

cd ArcShelf

npm install

npm run build
```

Load the generated **dist** folder as an unpacked extension.
